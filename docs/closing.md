# Closing & The Future

Grasswhistle started as a personal tool — a way to skip the tedious parts of region building and get straight to the fun stuff.

If you're reading this, it's hopefully doing that for you too.

This was a fun project, and I'm happy with how much of the original plan made it in. One thing I did have to cut was an in-app tile editor — I thought about it a lot, and eventually it became clear that as a solo developer, there was no realistic path to building something better than the tools already out there. [Aseprite](https://www.aseprite.org/) exists. Use Aseprite.

## Where things stand

As of V2, I consider Grasswhistle feature-complete. Everything I wanted to be here is here, and hopefully working well. I may come back to add things down the road — dungeon generation has been on my mind — but for now I'm happy with where it's landed.

If you run into a bug, please reach out to me on Reddit. And if you want to dig into the code, it's fully open on GitHub — you're welcome to fork and build on it, just give me a shoutout.

*PS: If you really want a challenge, try implementing water using the RPG Maker Autotile format. I couldn't wrap my head around it. Good luck.*
